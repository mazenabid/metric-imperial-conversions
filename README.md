A simple and interactive web app that instantly converts between metric and imperial units using a single input field.

Built with:
- HTML
- CSS
- JavaScript

## Features
- Real-time unit conversions
- Clean and responsive UI
- Lightweight and beginner-friendly
- Converts:
  - Meters ↔ Feet
  - Liters ↔ Gallons
  - Kilograms ↔ Pounds
